Curso de jQuery - Aula 05

Seletores de Filtros Básicos e Seletores de Conteúdo

Seletores de Filtros Básicos, por ordem no index

:first = Seleciona somente no primeiro elemento

:last = Seleciona somente no ultimo elemento

:not() = Não selecionara o item especificado como parametro

:even = Seleciona somente elementos de indice impar

:odd = Seleciona somente elementos de indice par

:eq() = Seleciona elemento de indice igual ao number informado como parametro

:gt() = Seleciona elementos de indice maiores que o numero informado como parametro

:lt() = Seleciona elementos de indice menores que o numero informado como parametro

Seletores de conteúdo

:contains() = Seleciona todos os elementos que contenha o texto informado no parametro

:has() = Seleciona todos os elementos que possuam outro elemento dentro de si de acordo com o parametro

:empty = Seleciona todos os elementos que não tem filhos, incluido textnodes

:parent = Seleciona todos os elementos que possuem filhos, incluido textnodes
