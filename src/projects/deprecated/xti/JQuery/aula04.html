Curso de jQuery - Aula 04

Seletores de Hierarquia, Seletores compostos

Exemplos de seletores

Os seletores funcionam básicamente conforme os seletores CSS

$("parent > child")

Seleciona todos os elementos filhos de um determinado elemento pai


$("antecentende descendente")

Seleciona um elemento descendente de um determinado antecentende


$("prev + next")

Seleciona todos os elementos próximos ao um elemento prévio


$("prev ~ sibling")

Seleciona todos os elementos irmãos depois do elemento prévio
