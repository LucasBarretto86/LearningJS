Eventos de Teclado, Input e Fomulários

Eventos de Teclado

.keyup() = Dispara quando uma tecla é solta.

.keyDown() = Dipera enquando uma tecla está pressionada.

.keypress() = Dispera quando uma tecla é pressionada e solta.

Eventos de Input e Formulário

.focus() = Dispara quando o elemento em questão está em foco.

.blur() = Dispara quando o elmento em questão perde o foco.

.change() = Evento disparado quando o elemento sofre alteração, usado para tag 'select'.

.submit() = Evento disparado a partir da submissão de um formulário, usado para validação de dados.

.select() = Evento disparado quando texto é seleionado
