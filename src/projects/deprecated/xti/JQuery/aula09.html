Curso de jQuery - Aula 09

Manipulação de Conteúdo

.html() = Permite o get e o set do conteúdo html presente dentro de um determinado elemento.

.text() = Permite o get e o set somente do texto presente dentro de um determinado elemento.

.val() = Retorna ou modifica o atributo value de um tag type input.

.replaceWith() = Função que permite a substituição de conteúdo, podendo receber html.
