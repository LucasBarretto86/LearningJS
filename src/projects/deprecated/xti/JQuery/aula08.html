Curso de jQuery - Aula 08

Manipulação de Atributo

.attr() = Recebe como parametro o nome do atributo e o novo valor, modificando os valores iniciais

.removeAttr() = Recebe o atributo como parametro e remove o mesmo das propriedades da tag

.hasClass() = Verifica se a tag possui uma determinada class

.addClass() = Adiciona uma class CSS para a tag

.removeClass() = Remove uma class CSS para a tag

.toggleClass() = Adiciona ou remove uma determinada class CSS a uma tag
