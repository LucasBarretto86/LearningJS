Manipulação CSS

Edição, Manipulação

.css() = Função que permite a atribuição de propriedades CSS como também retorna o valor do atributo. Para atribuir valor devera ser informado o nome da propriedade e o valor a atribuir, para retornar o valor somente o nome da propriedade deve ser informado.

.hasClass() = Função que retorna true ou false caso uma determinada case esteja presente no elemento.

.removeClass() = Função que remove uma classe que esteja atrelada ao elemento.

.addClass() = Função que permite a vinculação de uma classe ao elemento.

Dimensões

.width() = Retorna a lagura do elemento.

.innerWidth() = Retorna a lagura do elemento + padding.

.outerWidth() = Retorna a lagura do elemento + padding + border.

.outerWidth(true) = Retorna a lagura do elemento + padding + border + margin.

.height() = Retorna a altura do elemento.

.innerHeight() = Retorna a altura do elemento + padding.

.outerHeight() = Retorna a altura do elemento + padding + border.

.outerHeight(true) = Retorna a altura do elemento + padding + border + margin.

Posicionamento

.position().top = Retorna o valor Y do canto superior do elemento.

.position().left = Retorna o valor X do canto esquerdo do elemento.

.offset().top = Retorna o valor Y da margem de impressão do canto superior do elemento.

.offset().left = Retorna o valor X da margem de impressão do canto esquerdo do elemento.
