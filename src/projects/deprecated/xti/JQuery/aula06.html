Seletores de Atributos

Os seletores de atributo utilização [] para receber o paramentro, ou seja o atributo que será buscado, no entando,
também é possível buscar conforme o conteudo do atributo:


[attr] = Seleciona todos os elementos que possuam um determinado atributo independente do valor atribuido

[attr1][attr2] = Seleciona todos os elementos que possuam os atributos especificados, atributos multiplos

[attr=value] = Seleciona todos os elementos que possuam um determinado atributo e valor atribuido

[attr|=value] = Seleciona todos os elementos que contenha um atributo com prefixo desde que seguido por um hyfen (-)

[attr$=value] = Seleciona todos os elementos que possuam um atributo terminado exatamente por um certo valor

[attr*=value] = Seleciona todos os elementos que contenham um atributo com o valor especificado

[attr^=value] = Seleciona todos os elementos aonde o valor do atributo se inicie pelo valor especificado

[attr!=value] = Seleciona todos os elementos que não possuam um atributo como valor especificado
