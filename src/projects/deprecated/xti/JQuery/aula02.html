Filosofia

Selecionar
Refere-se exatamente ao modelo de selecionar os elementos, atraves do seletor $() de modo a simplificar o processo.

Encadear
Trata-se da possibilidade de atrelar diversas instruções em apenas uma linha de código, tenho em vista que cada função
dentro do jQuery retorna o proprio objeto.

Encapsular
Refere-se ao modole de manipulação dos elementos da DOM API de modo simplificado, ou seja, encapsulado pelas
funcionalidades da jQuery.

Iterar Impliciamente
Devido a possibilidade de iterar com elementos em array sem a necessidade de um for ou foreach.
