Seletores de Formulários

:input = Seleciona todos os elementosw input

:text = Seleciona todos os elementos de input com type text

:password = Seleciona todos os elementos de input com type password

:radio = Seleciona todos os elementos de input com type radio

:checkbox = Seleciona todos os elementos de input com type checkbox

:file = Seleciona todos os elementos do type file

:submit = Seleciona todos os elementos do type submit

:button = Seleciona todos os elementos button

:reset = Seleciona todos os elementos do type reset

:image = Seleciona todos os elementos de image

:enabled = Seleciona todos os elementos de input que não esterjam disabled

:disabled = Seleciona todos os elementos que estejam disabled

:checked = Seleciona todos os elementos selecionados/checked

:selected = Seleciona todos os elementos marcados como selected, uma funcionalidade mais especifica do que o checked

:focus = Seleciona todos os elementos que estejam em foco
